# Lefortech Landing Page

Sitio web institucional para **Lefortech**, una empresa de soporte IT y mantenimiento informático para PyMEs.

## Tecnologías

*   **HTML5**: Estructura semántica.
*   **CSS3**: Diseño responsive con "Tech Blue" theme, flexbox y grid.
*   **JavaScript (Vanilla)**: Menú móvil y animaciones de scroll.

## Estructura del Proyecto

*   `/css`: Archivos de estilos (style.css).
*   `/hero`: Componentes y lógica principal.
*   `/assets`: Imágenes y recursos estáticos.
*   `index.html`: Página principal.
